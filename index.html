In JavaScript, all values have a truthy or falsy value.

All variables that have been declared and assigned are truthy unless they contain one of the six values listed below:

  • false
  • 0 and -0
  • “” and ‘’ (empty strings)
  • null
  • undefined
  • NaN (Not a Number)
