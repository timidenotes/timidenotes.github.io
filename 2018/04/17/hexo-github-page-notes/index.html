hexo + github page = notes

發表於 2018-04-17

github pages

  1. 申請githbu帳號
  2. 建立repository, 名稱是“username.githubt.io”
  3. 上傳一個index.html
  4. https://username.github.io

hexo

  1. 安裝(前置：node.js、git)
  1. cd到github page的目錄下(原有資料即可跳過此步)
  1. 設定(_config.yml)
  1. 發布(clean很重要)

github ssh

  1. brew install git (macOS)
  1. 生成ssh密鑰文件、複製
  1. github settings新增ssh key

git command line

github desktop 和 git 可以分別使用不同的帳號 不會互相影響
可以先把網頁創好 push上github 之後再deploy
